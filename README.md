# Kafka Hello World

Run
- `docker-compose up -d`
- `docker-compose ps`
- `docker-compose exec goapp bash`
- `go mod tidy`
- `go run main.go`

Open browser
- `http://localhost:8080/ui`

Click on Topics
- Check the `test` topic message value sent by Kafka producer.
